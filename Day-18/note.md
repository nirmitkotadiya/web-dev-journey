## Block Elements
Block elements create a block or start on a new line.

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
```

### Output:
```
Heading 1
Heading 2
```

## Inline Elements
Inline elements do not create a new line; they remain in the same line as other inline elements.

```html
<a>Click here</a>
<a>Click here</a>
```

### Output:
```
Click here Click here
```
