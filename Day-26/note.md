## 🚀 Day 26: Text Size  

### 🔤 **Relative vs Absolute Sizing**  

| **Relative** (Flexible) | **Absolute** (Fixed) |  
|------------------------|----------------------|  
| Related to parent element | Independent of parent |  
| Uses `%`, `em`, `rem` | Uses `px` |  

### 📏 **Font Size Calculation**  
- **Common multipliers for `em` or `rem`** (assuming `16px` default size):  
  - `1.25em` → **20px**  
  - `1.5em` → **24px**  
  - `2em` → **32px**  
  - `3em` → **48px**
